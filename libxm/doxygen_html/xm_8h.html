void xm_contract ( xm_scalar_t  alpha,
const xm_tensor_t *  a,
const xm_tensor_t *  b,
xm_scalar_t  beta,
xm_tensor_t *  c,
const char *  idxa,
const char *  idxb,
const char *  idxc 
)

Contract tensors over contraction indices specified by strings idxa and idxb (c = alpha * a * b + beta * c). Permutation of tensor c is specified by idxc. The routine will perform optimal contraction using symmetry and sparsity information obtained from tensors' block-structures. Tensors must be setup beforehand so that they have correct symmetries. This function does not change the original block-structure of the output tensor.

Parameters
alphaScalar factor.
aFirst tensor.
bSecond tensor.
betaScalar factor.
cThird tensor.
idxaIndices of a.
idxbIndices of b.
idxcIndices of c.
Example: xm_contract(1.0, a, b, 2.0, c, "abcd", "ijcd", "ijab");
c_ijab = a_abcd * b_ijcd + 2 * c_ijab
